# Interpreting Forex List Results

## Reading the Output

The octagon-agent returns categorized currency pair information:

| Currency Pair | Description/Reason for Activity |
|---------------|--------------------------------|
| EUR/USD | Most liquid pair, Eurozone and U.S. economic interdependence |
| GBP/USD | High liquidity, sensitive to U.K./U.S. macroeconomic data |
| USD/JPY | Japanese Yen safe-haven status, U.S. interest rates |

The data includes:
- Currency pair symbols
- Activity/liquidity description
- Key drivers

## Understanding Currency Pair Categories

### Major Pairs

| Characteristics | Description |
|-----------------|-------------|
| Definition | USD paired with major currencies |
| Examples | EUR/USD, GBP/USD, USD/JPY |
| Liquidity | Highest |
| Spreads | Tightest (0.5-2 pips) |
| Best for | Most traders |

### Cross Pairs

| Characteristics | Description |
|-----------------|-------------|
| Definition | Major currencies without USD |
| Examples | EUR/GBP, EUR/JPY, GBP/JPY |
| Liquidity | Moderate to high |
| Spreads | Wider (2-5 pips) |
| Best for | Specific views |

### Exotic Pairs

| Characteristics | Description |
|-----------------|-------------|
| Definition | Major + emerging market currency |
| Examples | USD/MXN, USD/ZAR, USD/TRY |
| Liquidity | Lower |
| Spreads | Widest (5-50+ pips) |
| Best for | Specialized trading |

## Red Flags

### Pair Concerns

1. **Very wide spread** - High trading cost
2. **Low volume times** - Poor execution
3. **Political instability** - Unpredictable moves
4. **Intervention risk** - Central bank action

### Market Conditions

1. **News blackouts** - Reduced liquidity
2. **Holiday periods** - Thin markets
3. **Weekend gaps** - Risk exposure
4. **Flash crash risk** - Exotic pairs
