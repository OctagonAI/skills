# Industry P/E Ratios Skill

## Purpose

Retrieve industry-specific P/E ratios for peer group comparisons.

## When to Use

- Comparing company P/E to direct peers
- Understanding industry valuation norms
- Identifying relative value within industry
- Peer group analysis

## Example Prompt

```
Retrieve the latest industry P/E ratios for 2025-02-03, filtered by exchange NYSE and industry Semiconductors.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Industry | Industry name |
| Exchange | Stock exchange |
| P/E Ratio | Industry P/E |
| Date | As-of date |
