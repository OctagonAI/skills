# Skill: SEC 10-K Analysis

## Purpose

Analyze 10-K annual filings for public companies to extract key financial metrics, risk factors, and business insights.

## When to Use

- Due diligence on potential investments
- Understanding company risk profile
- Extracting audited financial data
- Analyzing management discussion and outlook
- Reviewing legal and regulatory disclosures

## Example Prompt

```
Analyze the latest 10-K filing for AAPL and extract key financial metrics and risk factors.
```

## Key Metrics Returned

| Metric | Description |
|--------|-------------|
| Total Revenue | Annual revenue from filing |
| Net Income | Annual profit |
| Total Assets | Balance sheet assets |
| Total Liabilities | Balance sheet liabilities |
| Operating Cash Flow | Cash from operations |
| Risk Factors | Key business risks disclosed |
| Source Citations | Page references to filing |

## Integration with Other Skills

- **Pairs with**: income-statement, balance-sheet for real-time data
- **Complements**: analyst-estimates for forward expectations
- **Supports**: Investment thesis with risk assessment

## Report Section Mapping

| Report Section | Usage |
|----------------|-------|
| Key Risks | Risk factor extraction |
| Investment Thesis | Business overview context |
| Appendix | Detailed filing analysis |
