# Sector P/E Ratios Skill

## Purpose

Retrieve sector P/E ratios by exchange for valuation benchmarking.

## When to Use

- Comparing company P/E to sector
- Understanding sector valuation levels
- Identifying relative value
- Sector rotation analysis

## Example Prompt

```
Retrieve the latest sector P/E ratios for 2025-02-03, filtered by exchange NASDAQ and sector Technology.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Sector | Sector name |
| Exchange | Stock exchange |
| P/E Ratio | Sector P/E |
| Date | As-of date |
