# Price Target Summary Skill

## Purpose

Retrieve analysts' price target summary with trends across timeframes.

## When to Use

- Tracking analyst sentiment trends
- Comparing recent vs. historical targets
- Evaluating consensus direction
- Identifying sentiment shifts

## Example Prompt

```
Retrieve the analysts' price-target summary for the stock symbol AAPL.
```

## Key Information Returned

| Timeframe | Data |
|-----------|------|
| Last Month | Number of analysts, average target |
| Last Quarter | Number of analysts, average target |
| Last Year | Number of analysts, average target |
| All Time | Number of analysts, average target |

## Integration with Other Skills

| Combine With | For |
|--------------|-----|
| price-target-consensus | Current targets |
| stock-quote | Target vs. price |
| stock-grades | Ratings context |
