# Batch Market Cap Skill

## Purpose

Retrieve market capitalization data for multiple companies at once for comparison.

## When to Use

- Comparing valuations across peers
- Ranking companies by size
- Portfolio composition analysis
- Competitive positioning

## Example Prompt

```
Retrieve market capitalization data for the following companies: AAPL, MSFT, GOOGL.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Company | Company name |
| Ticker | Stock symbol |
| Market Cap | Current market value |
| Ranking | Relative position |

## Integration with Other Skills

| Combine With | For |
|--------------|-----|
| company-market-cap | Individual detail |
| stock-price-change | Relative performance |
| sector-pe-ratios | Valuation comparison |
