# Skill: SEC Amendments Review

## Purpose

Review amendments to SEC filings and identify material changes or corrections.

## When to Use

- Checking for restatements
- Identifying corrections
- Tracking S-1 amendment progression
- Monitoring filing quality

## Example Prompt

```
Review recent amendments to SEC filings for AMZN and identify material changes or corrections.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| Amendment Type | 10-K/A, 8-K/A, etc. |
| Change Description | What was amended |
| Materiality | Significance of change |
| Timing | When filed |

## Integration with Other Skills

- **Pairs with**: All filing skills for context
- **Feeds into**: Filing Inventory, Material Events
- **Validates**: Filing quality and accuracy

## Report Section Mapping

| Report Section | Usage |
|----------------|-------|
| Executive Summary | Amendment flags |
| Appendix | Amendment history |
| Risk Assessment | Restatement risk |
