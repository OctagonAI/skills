# Earnings Competitive Review Skill

## Purpose
Analyze competitive landscape, market positioning, and strategic differentiation.

## When to Use
- Understanding competitive dynamics
- Market share commentary
- Differentiation analysis

## Example Prompt
```
Analyze management's discussion of competitive landscape and market positioning from AMD's earnings call.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Segment Analysis | Position by business unit |
| Key Differentiators | Competitive advantages |
| Competitor References | Direct and indirect mentions |
| Market Share | Position and trends |

## Report Section Mapping
→ Strategic Updates section
→ Competitive Positioning
