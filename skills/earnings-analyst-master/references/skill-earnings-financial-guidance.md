# Earnings Financial Guidance Skill

## Purpose
Extract financial guidance and forward-looking statements with segment detail and risk factors.

## When to Use
- Getting detailed financial guidance
- Segment-level projections
- Understanding guidance caveats

## Example Prompt
```
Extract and analyze financial guidance and forward-looking statements from MSFT's latest earnings transcript.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Revenue Guidance | Dollar range and growth rate |
| Segment Guidance | Business unit breakdown |
| Margin Guidance | Gross and operating margin targets |
| Risk Factors | Caveats and assumptions |
