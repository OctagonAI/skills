# Skill: SEC Footnotes Analysis

## Purpose

Analyze footnotes and accounting policies from SEC filings.

## When to Use

- Understanding accounting treatment
- Identifying critical estimates
- Finding off-balance sheet items
- Assessing revenue recognition policies

## Example Prompt

```
Analyze the footnotes and accounting policies from NVDA's latest quarterly filing.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| Accounting Policies | Significant policies |
| Critical Estimates | Judgment areas |
| Commitments | Off-balance sheet |
| Related Parties | Insider transactions |

## Integration with Other Skills

- **Pairs with**: sec-10k-analysis for context
- **Feeds into**: Financial Statement Analysis section
- **Validates**: Financial statement quality
