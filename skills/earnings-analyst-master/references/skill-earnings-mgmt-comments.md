# Earnings Management Comments Skill

## Purpose
Extract management's commentary on specific topics with executive attribution.

## When to Use
- Topic-specific deep dives
- Executive quote extraction
- Strategy commentary

## Example Prompt
```
Extract management's commentary about product development from AAPL's latest earnings call.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Topic Summary | High-level position |
| Key Quotes | Executive statements with attribution |
| Strategic Insights | Underlying strategy |
| Follow-up Questions | Deeper research areas |

## Report Section Mapping
→ Management Commentary section
→ Strategic Updates section
