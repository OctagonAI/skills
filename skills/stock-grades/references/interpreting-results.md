# Interpreting Stock Grades Results

## Reading the Output

The octagon-agent returns analyst rating information including:

- Recent rating actions (upgrades, downgrades, maintains)
- Institution/analyst source
- Previous and new ratings
- Dates of rating changes
- Historical rating context

## Understanding Rating Actions

### Action Types

| Action | Meaning |
|--------|---------|
| Upgrade | Rating improved |
| Downgrade | Rating lowered |
| Maintain | No change, reaffirmed |
| Initiate | New coverage |
| Reiterate | Repeated with conviction |

### Action Significance

| Change | Significance |
|--------|--------------|
| Sell → Buy | Major bullish shift |
| Hold → Buy | Positive momentum |
| Buy → Buy (maintain) | Continued conviction |
| Buy → Hold | Concerns emerging |
| Hold → Sell | Significant downgrade |

## Decoding Rating Terms

### Bullish Ratings

| Term | Meaning |
|------|---------|
| Buy | Expect outperformance |
| Overweight | Above benchmark allocation |
| Outperform | Beat sector/market |
| Strong Buy | Highest conviction |
| Accumulate | Build position |

### Neutral Ratings

| Term | Meaning |
|------|---------|
| Hold | Maintain position |
| Neutral | No strong view |
| Market Perform | Match market returns |
| Equal-Weight | Benchmark allocation |
| Sector Perform | Match sector |

### Bearish Ratings

| Term | Meaning |
|------|---------|
| Sell | Expect underperformance |
| Underweight | Below benchmark |
| Underperform | Lag sector/market |
| Reduce | Decrease position |
| Avoid | Stay away |

## Analyzing Trends

### Upgrade Trends

| Pattern | Interpretation |
|---------|----------------|
| Single upgrade | One analyst bullish |
| Multiple upgrades | Broad sentiment shift |
| Major bank upgrade | Significant signal |
| Post-earnings upgrade | Results exceeded expectations |

### Downgrade Trends

| Pattern | Interpretation |
|---------|----------------|
| Single downgrade | One analyst cautious |
| Multiple downgrades | Growing concerns |
| Major bank downgrade | Red flag |
| Post-earnings downgrade | Disappointing results |

### Maintain Trends

| Pattern | Interpretation |
|---------|----------------|
| All maintains | Stable outlook |
| Maintain after news | No change in view |
| Long maintain streak | Consensus solidified |

## Rating Distribution Analysis

### Calculating Consensus

| Metric | Formula |
|--------|---------|
| Buy % | Buy ratings / Total ratings |
| Hold % | Hold ratings / Total ratings |
| Sell % | Sell ratings / Total ratings |

### Interpreting Distribution

| Distribution | Meaning |
|--------------|---------|
| >75% Buy | Strong bullish consensus |
| 50-75% Buy | Moderately bullish |
| Mixed (no majority) | Divided opinion |
| >50% Hold | Wait-and-see mode |
| >25% Sell | Notable bearish presence |

## Timing Analysis

### Rating Freshness

| Age | Status |
|-----|--------|
| Same day | Breaking news |
| <1 week | Very current |
| 1-4 weeks | Recent |
| 1-3 months | Reasonably current |
| >3 months | May be stale |

### Catalyst Timing

| Event | Rating Response |
|-------|-----------------|
| Earnings | Day of or +1 day |
| Guidance | Within 48 hours |
| Product launch | Same week |
| Macro news | As relevant |

| Action | Common Impact |
|--------|---------------|

| Multiple upgrades | Stronger positive |
| Major bank downgrade | Negative price move |
| Multiple downgrades | Stronger negative |

### Impact Factors

| Factor | Increases Impact |
|--------|------------------|
| Analyst reputation | Well-known analyst |
| Surprise factor | Unexpected change |
| Target change | Accompanies rating |
| Timing | First to call |

## Red Flags

### Warning Signs

1. **Sudden downgrades** - Investigate cause
2. **Coverage drops** - Declining interest
3. **Conflict of interest** - Banking relationships
4. **Lagging calls** - Following rather than leading
5. **No recent updates** - Stale coverage

### Data Quality Issues

| Issue | Concern |
|-------|---------|
| Old ratings | May not reflect current view |
| Unknown analysts | Limited credibility |
| Inconsistent terms | Hard to compare |
| Missing context | No reason for change |
