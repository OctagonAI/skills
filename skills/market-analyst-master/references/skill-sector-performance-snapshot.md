# Sector Performance Snapshot Skill

## Purpose

Retrieve sector-wide metrics including revenue, EBITDA, market cap, and enterprise value.

## When to Use

- Analyzing sector fundamentals
- Understanding sector scale
- Competitive landscape overview
- Sector health assessment

## Example Prompt

```
Retrieve a snapshot of market sector performance for 2025-02-03, filtered by exchange NASDAQ and sector Technology.
```

## Key Information Returned

| Metric | Description |
|--------|-------------|
| Revenue | Sector total revenue |
| EBITDA | Sector EBITDA |
| Net Income | Sector net income |
| Market Cap | Sector market cap |
| Enterprise Value | Sector EV |
| Employees | Sector headcount |
