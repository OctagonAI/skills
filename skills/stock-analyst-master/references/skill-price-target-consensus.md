# Price Target Consensus Skill

## Purpose

Retrieve consensus, median, high, and low analyst price targets.

## When to Use

- Calculating upside/downside potential
- Evaluating target range spread
- Understanding bull/bear cases
- Assessing analyst agreement

## Example Prompt

```
Retrieve consensus price targets for the stock symbol AAPL.
```

## Key Information Returned

| Metric | Description |
|--------|-------------|
| Consensus Target | Average of all targets |
| Median Target | Middle target value |
| High Target | Most bullish target |
| Low Target | Most bearish target |
| Spread | Range between high and low |

| Combine With | For |
|--------------|-----|
