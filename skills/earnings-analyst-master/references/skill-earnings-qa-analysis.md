# Earnings Q&A Analysis Skill

## Purpose
Analyze the Q&A section for strategic insights and analyst concerns.

## When to Use
- Understanding analyst focus areas
- Evaluating management responses
- Identifying unresolved issues

## Example Prompt
```
Analyze the Q&A section of AAPL's latest earnings call for insights about future strategy.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Strategic Themes | Major topics discussed |
| Key Insights | Important takeaways |
| Analyst Concerns | Issues raised by analysts |
| Response Quality | How well questions were answered |

→ Q&A Analysis section
→ Analyst Sentiment section
