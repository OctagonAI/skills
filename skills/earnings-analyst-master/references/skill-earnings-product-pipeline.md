# Earnings Product Pipeline Skill

## Purpose
Extract R&D updates, clinical trials, regulatory filings, and launch timelines.

## When to Use
- Pipeline valuation
- Catalyst tracking
- R&D progress monitoring

## Example Prompt
```
Extract product development and pipeline updates from MRNA's latest earnings call discussion.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Development Milestones | Key progress updates |
| Clinical Trials | Trial status and data timing |
| Regulatory Submissions | FDA/EMA filings |
| Launch Timeline | Commercialization dates |
| Portfolio Overview | Stage breakdown |
