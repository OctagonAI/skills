# Earnings Analyst Questions Skill

## Purpose
Identify key themes and concerns raised by analysts with attribution.

## When to Use
- Tracking analyst concerns
- Identifying recurring themes
- Understanding Street focus

## Example Prompt
```
Identify key themes and concerns raised by analysts during TSLA's latest earnings call.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Key Themes | Major topics raised |
| Analyst Attribution | Who asked what |
| Concern Categories | Grouped by topic area |
| Question Frequency | How often topics were asked |
