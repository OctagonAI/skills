# Skill: SEC Annual Comparison

## Purpose

Compare key metrics and risk factors between current and prior year 10-K filings.

## When to Use

- Tracking year-over-year changes
- Identifying disclosure evolution
- Risk factor change analysis
- Strategic shift detection

## Example Prompt

```
Compare key metrics and risk factors between CRM's current and previous year 10-K filings.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| Financial Changes | YoY metric comparison |
| Risk Changes | New, removed, modified risks |
| Disclosure Evolution | Changed language |
| Strategic Shifts | Emphasis changes |

## Integration with Other Skills

- **Pairs with**: sec-10k-analysis for detail
- **Feeds into**: YoY Comparison section
- **Validates**: Trends and changes
