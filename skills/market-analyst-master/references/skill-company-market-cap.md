# Company Market Cap Skill

## Purpose

Retrieve current market capitalization data for a single company with size classification.

## When to Use

- Getting current valuation snapshot
- Determining size classification (mega/large/mid/small-cap)
- Understanding market value context
- Quick valuation check

## Example Prompt

```
Get market capitalization data for the symbol AAPL.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Market Cap | Current total market value |
| Date | As-of date |
| Size Classification | Mega/Large/Mid/Small/Micro-cap |
