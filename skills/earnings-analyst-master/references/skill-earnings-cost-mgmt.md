# Earnings Cost Management Skill

## Purpose
Analyze cost reduction initiatives, restructuring, and operational efficiency.

## When to Use
- Tracking efficiency programs
- Restructuring analysis
- Margin improvement assessment

## Example Prompt
```
Analyze cost reduction initiatives and operational efficiency measures from META's earnings transcript.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Key Initiatives | Cost reduction programs |
| Workforce Actions | Hiring, layoffs, reorg |
| Facility Changes | Real estate optimization |
| Expected Savings | Quantified impact |
