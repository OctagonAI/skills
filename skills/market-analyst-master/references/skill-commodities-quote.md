# Commodities Quote Skill

## Purpose

Retrieve real-time commodity price quotes with day ranges and moving averages.

## When to Use

- Getting current commodity prices
- Analyzing commodity trends
- Understanding macro context
- Commodity-sensitive stock analysis

## Example Prompt

```
Retrieve the real-time price quote for GCUSD.
```

## Key Information Returned

| Metric | Description |
|--------|-------------|
| Current Price | Last traded price |
| Change | Daily change |
| Day Range | Low - High |
| 52-Week Range | Annual range |
| 50-Day MA | Short-term average |
| 200-Day MA | Long-term average |
| Volume | Trading volume |

## Common Symbols

| Symbol | Commodity |
|--------|-----------|
| GCUSD | Gold |
| SIUSD | Silver |
| CLUSD | Crude Oil |
| NGUSD | Natural Gas |
