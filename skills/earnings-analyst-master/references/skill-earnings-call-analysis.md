# Earnings Call Analysis Skill

## Purpose
Analyze earnings call transcripts with follow-up questions and source citations.

## When to Use
- Full transcript analysis
- Generating follow-up questions
- Getting source-cited insights

## Example Prompt
```
Analyze AAPL's latest earnings call transcript and extract key insights about future guidance.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Financial Guidance | All forward-looking statements |
| Strategic Focus | Key initiatives |
| Risk Factors | Challenges mentioned |
| Follow-up Questions | AI-generated research questions |
| Source Citations | Page references |

## Report Section Mapping
→ Guidance Analysis section
→ Management Commentary section
