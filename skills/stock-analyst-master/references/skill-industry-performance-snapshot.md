# Industry Performance Snapshot Skill

## Purpose

Retrieve daily industry performance overview with average price changes.

## When to Use

- Tracking industry daily moves
- Identifying sector rotation
- Comparing industry momentum
- Catalyst impact analysis

## Example Prompt

```
Retrieve a daily overview of industry performance for 2025-01-09, filtered by exchange NASDAQ and industry Biotechnology.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Industry | Industry name |
| Exchange | Stock exchange |
| Date | Trading date |
| Average Change | Industry daily return |

## Integration with Other Skills

| Combine With | For |
|--------------|-----|
