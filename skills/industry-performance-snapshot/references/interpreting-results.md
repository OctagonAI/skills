# Interpreting Industry Performance Snapshot Results

## Reading the Output

The octagon-agent returns daily industry performance data:

| Metric | Value |
|--------|-------|
| Industry | Biotechnology |
| Exchange | NASDAQ |
| Date | 2025-01-09 |
| Average Change | +7.89% |

## Understanding the Data

### What Average Change Means

| Aspect | Description |
|--------|-------------|
| Definition | Mean price change of stocks in industry |
| Scope | All companies in the industry |
| Timeframe | Single trading day |
| Basis | Close-to-close or intraday |

### Calculation Methods

| Method | Description |
|--------|-------------|
| Equal-Weighted | Each stock counts equally |
| Market-Cap Weighted | Large-caps have more influence |
| Median | Middle stock performance |

## Comparing to Market

### Relative Performance

| Calculate | Formula |
|-----------|---------|
| Alpha | Industry Return - Market Return |
| Relative Strength | Industry Return / Market Return |

### Example

If S&P 500 returned +0.5% and Biotech +7.89%:
- Alpha: +7.39%
- Relative Strength: 15.8x market
- **Interpretation**: Massive outperformance

### Interpretation Guide

| Alpha | Meaning |
|-------|---------|
| >3% | Significant outperformance |
| 1-3% | Moderate outperformance |
| 0-1% | Slight outperformance |
| 0% | In-line with market |
| -1 to 0% | Slight underperformance |
| <-3% | Significant underperformance |

## Breadth Considerations

### What to Ask

| Question | Why It Matters |
|----------|----------------|
| How many stocks up? | Broad vs. narrow move |
| Leaders vs. laggards | Concentration |
| Large-cap vs. small-cap | Size effect |

### Breadth Interpretation

| Breadth | Meaning |
|---------|---------|
| 80%+ up | Broad strength |
| 50-80% up | Selective strength |
| <50% up | Narrow, concentrated |

## Red Flags

### Warning Signs

1. **Huge move, no news** - May be manipulation or error
2. **Single stock dominance** - Not representative
3. **Reversal next day** - False signal
4. **Low volume move** - Lacks conviction
5. **Gap vs. sector** - Verify data accuracy
