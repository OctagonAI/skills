# Skill: SEC S-1 Analysis

## Purpose

Analyze S-1 registration statements for IPO risks, opportunities, and capitalization.

## When to Use

- IPO due diligence
- Pre-IPO company research
- Understanding cap table and ownership
- Assessing use of proceeds

## Example Prompt

```
Analyze the S-1 registration statement for Figma and extract key business risks and opportunities.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| Business Overview | Company description |
| Risk Factors | IPO-specific risks |
| Use of Proceeds | Planned fund allocation |
| Capitalization | Pre/post-IPO ownership |
| Financials | Historical performance |

## Integration with Other Skills

- **Pairs with**: sec-risk-factors for risk analysis
- **Feeds into**: IPO section of report
- **Validates**: Company claims with required disclosures

## Report Section Mapping

| Report Section | Usage |
|----------------|-------|
| IPO/Registration | Primary source |
| Risk Analysis | S-1 specific risks |
| Governance | Cap table, ownership |
