# Skill: SEC MD&A Analysis

## Purpose

Analyze Management Discussion and Analysis sections for strategic insights.

## When to Use

- Understanding management's view of results
- Identifying strategic priorities
- Assessing performance drivers
- Evaluating forward-looking statements

## Example Prompt

```
Analyze the Management Discussion and Analysis section from AAPL's latest quarterly report.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| Strategic Initiatives | Management priorities |
| Performance Drivers | Revenue and margin explanations |
| Macroeconomic Factors | Economic context |
| Forward Guidance | Management outlook |

## Integration with Other Skills

- **Pairs with**: sec-10k-analysis, sec-10q-analysis
- **Feeds into**: Business Overview section
- **Validates**: Financial results with management context

## Report Section Mapping

| Report Section | Usage |
|----------------|-------|
| Business Overview | Strategic priorities |
| Financial Analysis | Performance context |
| YoY Comparison | Changed emphasis |
