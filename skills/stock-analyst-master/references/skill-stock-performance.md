# Stock Performance Skill

## Purpose

Retrieve daily closing prices, trading volume, and performance metrics for public companies.

## When to Use

- Analyzing historical price trends
- Tracking daily trading activity
- Evaluating volume patterns
- Identifying support/resistance levels

## Example Prompt

```
Retrieve the daily closing prices for AAPL over the last 30 days.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Date | Trading day |
| Closing Price | End-of-day price |
| Volume | Shares traded |
| Price Trend | Direction over period |

## Integration with Other Skills

| Combine With | For |
|--------------|-----|
| stock-quote | Current vs. historical prices |
| stock-price-change | Returns vs. daily data |
| stock-historical-index | Stock vs. index comparison |
