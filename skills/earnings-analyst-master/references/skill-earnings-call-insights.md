# Earnings Call Insights Skill

## Purpose
Analyze earnings call transcripts to extract future guidance, strategic priorities, and management commentary.

## When to Use
- Getting high-level earnings takeaways
- Extracting forward-looking guidance
- Understanding strategic priorities

## Example Prompt
```
Analyze AAPL's latest earnings call transcript and extract key insights about future guidance.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Financial Outlook | Revenue, margin, EPS expectations |
| Strategic Highlights | Key initiatives and priorities |
| Risk Factors | Challenges and headwinds |
| Management Sentiment | Tone and confidence |

## Report Section Mapping
→ Executive Summary section
→ Guidance Analysis section
