# Skill: SEC Corporate Governance

## Purpose

Review corporate governance practices, board composition, and policies.

## When to Use

- Board quality assessment
- Governance policy review
- Related party analysis
- Shareholder rights evaluation

## Example Prompt

```
Review corporate governance practices and board composition from ORCL's latest proxy statement.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| Board Composition | Directors, independence |
| Committees | Structure and membership |
| Governance Policies | Key policies |
| Shareholder Rights | Voting, proxy access |

## Integration with Other Skills

- **Pairs with**: sec-proxy-analysis for compensation
- **Feeds into**: Governance Assessment section
- **Validates**: Governance quality
