# Earnings Capital Allocation Skill

## Purpose
Extract capital allocation, investment priorities, buybacks, and dividend commentary.

## When to Use
- Understanding investment priorities
- Tracking shareholder returns
- M&A appetite assessment

## Example Prompt
```
Extract management's commentary on capital allocation and investment priorities from GOOGL's earnings transcript.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| CapEx | Capital expenditure plans |
| Buybacks | Share repurchase activity |
| Dividends | Dividend policy |
| M&A | Acquisition strategy |
| Investment Focus | Priority areas |
