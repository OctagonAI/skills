# Earnings Conference Call Sentiment Skill

## Purpose
Analyze overall sentiment and tone of management during conference calls.

## When to Use
- Gauging management confidence
- Tracking sentiment changes
- Language pattern analysis

## Example Prompt
```
Analyze the overall sentiment and tone of management during NVDA's latest earnings conference call.
```

## Key Information Returned
| Data Point | Description |
|------------|-------------|
| Overall Sentiment | Optimistic/neutral/cautious |
| Confidence Level | High/moderate/low |
| Key Themes | Sentiment drivers |
| Language Patterns | Positive vs. cautionary |
