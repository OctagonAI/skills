# Forex List Skill

## Purpose

Retrieve listing of actively traded currency pairs (majors, crosses, exotics).

## When to Use

- Researching forex markets
- Understanding currency exposure
- Analyzing macro factors
- Multi-asset context

## Example Prompt

```
Retrieve a full listing of actively traded currency pairs in the global forex market.
```

## Key Information Returned

| Category | Examples |
|----------|----------|
| Major Pairs | EUR/USD, GBP/USD, USD/JPY |
| Cross Pairs | EUR/GBP, EUR/JPY, AUD/NZD |
| Exotic Pairs | USD/MXN, USD/ZAR, USD/TRY |
| Key Drivers | Economic factors for each pair |
