# Skill: SEC Proxy Analysis

## Purpose

Extract executive compensation and governance info from proxy statements.

## When to Use

- Executive compensation analysis
- Pay-for-performance assessment
- Shareholder voting research
- Say-on-pay evaluation

## Example Prompt

```
Extract executive compensation details and governance information from TSLA's latest proxy statement.
```

## Key Information Returned

| Section | Content |
|---------|---------|
| NEO Compensation | Executive pay details |
| Pay Components | Salary, bonus, equity |
| Voting Matters | Proposals and results |
| Shareholder Proposals | Investor initiatives |

## Integration with Other Skills

- **Pairs with**: sec-corp-governance for full governance view
- **Feeds into**: Governance Assessment section
- **Validates**: Compensation alignment

## Report Section Mapping

| Report Section | Usage |
|----------------|-------|
| Governance Assessment | Compensation |
| Risk Assessment | Pay misalignment |
| Executive Summary | Key governance issues |

octagon-sec-agent, octagon-companies-agent, octagon-web-search-agent
