## Understanding the Metrics

### Revenue

| Aspect | Description |
|--------|-------------|
| What it is | Total sales/income |
| Timeframe | Trailing 12 months (TTM) |
| Use | Size and scale indicator |
| Compare to | Prior periods, peers |

### EBITDA

| Aspect | Description |
|--------|-------------|
| What it is | Earnings Before Interest, Taxes, Depreciation, Amortization |
| Meaning | Operating profitability |
| Use | Core business performance |
| Margin | EBITDA / Revenue |

### Net Income

| Aspect | Description |
|--------|-------------|
| What it is | Bottom-line profit |
| Meaning | Earnings after all expenses |
| Use | Ultimate profitability |
| Margin | Net Income / Revenue |

### Market Cap

| Aspect | Description |
|--------|-------------|
| What it is | Share price Ã— shares outstanding |
| Meaning | Equity market value |
| Use | Valuation, size classification |
| Compare to | Revenue, earnings |

### Enterprise Value

| Aspect | Description |
|--------|-------------|
| What it is | Market Cap + Debt - Cash |
| Meaning | Total firm value |
| Use | M&A, valuation multiples |
| Compare to | EBITDA (EV/EBITDA) |
