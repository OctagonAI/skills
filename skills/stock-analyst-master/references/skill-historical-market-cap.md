# Historical Market Cap Skill

## Purpose

Retrieve historical market capitalization data over specified date ranges.

## When to Use

- Tracking valuation trends over time
- Identifying peak and trough valuations
- Analyzing market cap growth
- Calculating drawdowns

## Example Prompt

```
Retrieve historical market capitalization data for AAPL from 2024-01-01 to 2025-01-31, limited to 1000 records.
```

## Key Information Returned

| Data Point | Description |
|------------|-------------|
| Date | Trading day |
| Market Cap | Daily market cap value |
| High | Peak in period |
| Low | Trough in period |
| Trend | Direction over time |

## Integration with Other Skills

| Combine With | For |
|--------------|-----|
| company-market-cap | Current vs. historical |
| stock-performance | Price driving cap changes |
| stock-price-change | Returns vs. cap growth |
